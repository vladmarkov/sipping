$ python sipping.py -d sip.test.info -p 5060 -i 192.168.1.196 -c 3 10.10.10.148

The above command sends OPTIONS pings to 10.10.10.148 for 3 times on port 5060 coming from this PC's local IP 192.168.1.196

  -x [X]       Print raw transmitted packets
  -X [X]       Print raw received responses
  -u userid    User part of the From header (default sipping)
  -d domain    Domain part of the From header (needed if your device filters based on domain);e.g., sip.test.info
  -p port      Destination port (default 5060)
  -i ip        IP to send in the Via header (will TRY to get local IP by default);e.g., 192.168.1.196

  Use -h for help
